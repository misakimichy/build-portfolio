Bookshelf

HTML & CSS design and build websites by Jon Duckett

The design of the book is really beautiful. I recommend this book for someone who just started learning or someone who want to brush up HTML and CSS knowledge.

Responsive Web Design 2nd Edition by Chris Ward

The concept of responsive design is really powerful and it's necessary to understand it as a front-end web developer.

The Linux Command Line: A Complete Introduction by William E. Shotts. Jr.

This book is easy for people who are even new to the command line. It's easy to find command line instructions on YouTube or tech blogs nowadays but it's worth reading. Let me say that this book is more like a learning book not a reference book.

Above the Fold: Understanding the Principles of Successful Web Site Design by Brian D. Miller

If you are studying web design or studying about font-end web developer, this book is helpful.
